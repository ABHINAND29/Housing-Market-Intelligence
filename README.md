### Housing-Market-Intelligence
This project presents an interactive Power BI dashboard designed to analyze housing market trends across Bangalore. It provides insights into builder performance, booking behavior, customer preferences, and sales channel effectiveness.
The dashboard enables users to explore data dynamically using filters and visualize key business metrics for better decision-making.

🎯 Objectives
Analyze booking conversion rates across micro-markets
Identify top-performing builders based on revenue and booking success
Understand customer preferences by configuration and buyer type
Evaluate sales channel effectiveness
Track luxury housing trends over time

🧱 Dataset Features

The dataset includes:

Property details (size, configuration, pricing)
Builder information
Booking status
Buyer type (Investor / End-user)
Sales channel
Micro-market location
Amenity and connectivity scores

⚙️ Key Features & Visualizations
📌 KPI Indicators
Total Properties
Booked Properties
Booking Conversion Rate (%)
Total Revenue
Average Ticket Price

🌍 Geographic Analysis
Map visualization showing property distribution across micro-markets
Highlights high-demand and premium areas

📈 Trend Analysis
Line chart tracking luxury housing bookings across quarters
Identifies growth patterns and seasonal demand

🏗️ Builder Performance
Bar chart and matrix showing builder-wise revenue and quarterly dominance
Ranking of top builders

🏘️ Demand Analysis
Donut chart showing most in-demand housing configurations (1BHK–4BHK)

📊 Conversion Analysis
100% stacked charts for:
Micro-market booking success
Sales channel performance

👥 Buyer Behavior Insights
Clustered charts analyzing:
Possession status vs booking decisions
Buyer type preferences

🔍 Key Insights
3BHK configurations dominate market demand
Ready-to-move properties show higher booking conversion
Broker channels contribute significantly to bookings
Premium micro-markets drive luxury housing demand
Builder performance varies across quarters, indicating competitive dynamics

🛠️ Tools & Technologies
Power BI – Data visualization & dashboarding
DAX – Calculated measures and columns
Data Modeling – Relationship building and transformation

📌 Conclusion

This dashboard provides a comprehensive view of the housing market, enabling stakeholders to identify trends, optimize strategies, and make data-driven decisions.
